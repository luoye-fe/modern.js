#官方插件

  • @modern-js/plugin-module-import：使用 SWC 提供与 babel-plugin-import 一样的能力。
  • @modern-js/plugin-module-banner：为每个 JS 和 CSS 文件的顶部和底部添加自定义内容，例如版权信息。
  • @modern-js/plugin-module-node-polyfill：会自动注入 Node 核心模块在浏览器端的 polyfills。
